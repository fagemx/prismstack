# Prismstack Wave 3 — Persistent Service Skills

**Goal:** Add 4 persistent service skills: `/source-convert`, `/tool-builder`, `/domain-upgrade`, `/workflow-edit`.

---

## Task 1: /source-convert
Convert external sources (repo, prompt, video, article, book, SOP, ECC skill, git history, idea) into skill content.

## Task 2: /tool-builder
Build tool-type skills (browser automation, API, CLI, file processing). Dual-layer: hands-on mode + meta mode (produce skill that can do it).

## Task 3: /domain-upgrade
Persistent upgrade workflow: listen to needs, test feedback, iterate. Orchestrates /skill-edit, /source-convert, /tool-builder, /workflow-edit.

## Task 4: /workflow-edit
View/edit artifact flow, skill connections, workflow graph.
