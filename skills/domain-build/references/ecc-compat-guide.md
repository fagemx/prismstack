# ECC 格式相容性規則

> 確保 /domain-build 生成的 skill 與 ECC（Claude Code skill 系統）完全相容。

---

## YAML Frontmatter 必要欄位

每個 SKILL.md 的 YAML frontmatter **必須**包含：

```yaml
---
name: {skill-name}
version: 0.1.0
origin: prismstack-generated
description: |
  {觸發條件 + 反觸發條件}
---
```

### 欄位說明

**`description`** — ECC 掃描此欄位進行 skill routing。必須包含：
- 做什麼（一句話）
- Trigger: 何時用（具體場景 + 用戶常說的話）
- Do NOT use when: 何時不用（具體場景 + 應該用什麼替代）

**`origin: prismstack-generated`** — 標識來源，區分手寫 skill 和自動生成 skill。

**`version: 0.1.0`** — 初始版本。遵循 semver。

---

## 安裝路徑

Skills 安裝到：`~/.claude/skills/{domain}/`

```
~/.claude/skills/
├── {domain}/                  ← domain gstack 安裝到這裡
│   ├── shared/
│   ├── routing/
│   │   └── SKILL.md
│   ├── {skill-1}/
│   │   └── SKILL.md
│   └── ...
└── {other-skills}/            ← 不衝突：每個 domain 有自己的子目錄
```

**不會與 ECC 的 `~/.claude/skills/` 中的其他 skill 衝突。**

---

## 檔案結構慣例

- 每個 skill 目錄的根目錄放 **`SKILL.md`**（ECC 慣例）
- 不要用 `SKILL.md.tmpl` — 那是 gstack template engine 的格式，ECC 不認
- 不要用 `index.md` 或 `main.md` — ECC 預期 `SKILL.md`

```
{skill-name}/
├── SKILL.md              ← ECC 掃描入口（必須）
├── references/           ← 選用，SKILL.md 引用
│   ├── gotchas.md
│   └── ...
└── scripts/              ← 選用，helper code
    └── ...
```

## 禁止事項

### 1. 不要使用 `{{PLACEHOLDER}}` 語法

```
❌ {{domain_name}}
❌ {{user_name}}
❌ {{slug}}
```

`{{}}` 是 gstack template engine 的語法。ECC 不會處理模板變數。
SKILL.md 必須是**可直接讀取的 markdown**，不需要預處理。

生成時直接寫入具體值：
```
✅ ~/.gstack/projects/game-dev/
✅ origin: prismstack-generated
```

### 2. 不要依賴 gstack runtime

```
❌ 假設 gstack CLI 已安裝
❌ 調用 gstack 專屬命令
❌ 依賴 gstack template rendering
```

生成的 skill 必須在**純 Claude Code 環境**下可用。
唯一的依賴是：`~/.gstack/projects/` 目錄（用於 artifact 存儲，skill 自行建立）。

### 3. 不要在 description 中放非觸發資訊

```
❌ description: "This skill was generated by prismstack on 2026-03-25"
❌ description: "Version 0.1.0 of the game review skill"
```

description 是 routing rule，不是 metadata。Metadata 放其他 frontmatter 欄位。

---

## 自檢清單

生成每個 skill 後，檢查：

- [ ] YAML frontmatter 有 `description:` 且包含觸發條件
- [ ] YAML frontmatter 有 `origin: prismstack-generated`
- [ ] YAML frontmatter 有 `version: 0.1.0`
- [ ] 檔名是 `SKILL.md`（不是 `.tmpl`）
- [ ] 沒有 `{{}}` 佔位符
- [ ] 不依賴 gstack CLI
- [ ] artifact 路徑用具體 slug（不是模板變數）
- [ ] description 的第一句是做什麼，不是 metadata
