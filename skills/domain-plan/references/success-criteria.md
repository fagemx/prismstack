# Skill Map Quality Score

5 dimensions, 0-2 each, 10 points max.

| Dimension | 0 | 1 | 2 |
|-----------|---|---|---|
| **Coverage** | <50% lifecycle stages covered | 50-80% covered | >80% lifecycle stages have skills |
| **Type Balance** | >70% one type (e.g. all Review) | 2-3 types represented | All 5 types present where appropriate |
| **Independence** | >3 skills fail independence test | 1-2 fail | All pass 3 tests |
| **Bridge Layer** | No bridge skills between design→production | Bridge exists but thin | Complete bridge with clear input/output |
| **Artifact Flow** | >3 orphan artifacts | 1-2 orphans | Every artifact has producer + consumer |

**Grade:** 8-10=Ship-ready, 5-7=Usable, 2-4=Needs work, 0-1=Restart
